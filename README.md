# ETL FACTURAS

Proyecto para procesamiento de facturas 

## Resumen

Este repositorio contiene un script principal (`main.py`) y utilidades en `funciones.py` y `prompt.py` para procesar facturas. Coloca tus facturas en la carpeta `facturas/` según indica el archivo ahí presente.

## Estructura del proyecto

- `main.py` - punto de entrada principal.
- `funciones.py` - funciones utilitarias del proyecto.
- `prompt.py` - prompts o plantillas usadas por el proyecto.
- `facturas/` - carpeta donde debes poner tus facturas (PDF o .txt).
